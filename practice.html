10) Practice Questions

Requirements
  1. Create a new HTML file named practice.html. Add the basic page structure with a suitable title.
  2. Inside the body, add a main heading h1 that says “My First Web Page
  3. Add two paragraphs describing what you want to learn in this unit. Put a line break in the first paragraph.
  4. Create an unordered list with three items: one hobby, one favourite food, and one programming goal.
  5. Add a link to any website you like. Make the link text meaningful (not “click here”).
  6. Insert an image that is saved in the same folder as your HTML file. Include a clear alt description.
  7. Use special characters to write: &, <, and > in a sentence.
  8. Place a comment above your heading explaining what the page is about.
